# Blinkit Grocery Sales Dashboard | Microsoft Excel

A data analysis and visualization project built on Blinkit's grocery retail dataset, covering 8,500+ transactions across multiple outlet types, tiers, and product categories from 2012 to 2022.

Project Overview:
This project involved end-to-end data analysis — from raw data cleaning to an interactive Excel dashboard to uncovering sales trends, outlet performance, and product-level insights for a quick-commerce grocery platform.

---

Key Steps:
1) Data Cleaning: Standardized inconsistent values in the `Item Fat Content` column (e.g., "Low fat" vs "Low Fat") and handled missing `Item Weight` entries (~17% of records) using mean imputation by item type.
2) Analysis: Calculated business KPIs — Total Sales (₹12L+), Average Order Value (₹140.99), No. of Orders (8,523), and Average Rating (3.97).
3) Dashboard: Built an interactive dashboard with slicers (Outlet Size, Year, Outlet Type) and 6+ visualizations including bar charts, donut charts, and a sales trend line (2012–2022).

---
 
Dataset Columns:
`Item Fat Content` · `Item Type` · `Item Identifier` · `Outlet Type` · `Outlet Size` · `Outlet Location Type` · `Outlet Establishment Year` · `Item Visibility` · `Item Weight` · `Sales` · `Rating`

---

Tools Used:
Microsoft Excel: Pivot Tables, Slicers, Pivot Charts, Power Query
